1. **Q: Which algorithm performed best?**
   **A:** Random Forest (check your notebook results for exact accuracy)

2. **Q: Explain Random Forest**
   **A:** Ensemble of decision trees, each votes for a class, majority wins

3. **Q: What is overfitting?**
   **A:** Model learns training data too well including noise, poor generalization

4. **Q: Why use train-test split?**
   **A:** To evaluate model on unseen data and prevent overfitting

5. **Q: What is PCA?**
   **A:** Principal Component Analysis - reduces dimensions while preserving variance

6. **Q: Difference between supervised and unsupervised?**
   **A:** Supervised uses labeled data, unsupervised finds patterns without labels

7. **Q: Why use Flask?**
   **A:** Lightweight, easy to learn, good for small-medium apps, Python integration

8. **Q: What is Naive Bayes?**
   **A:** Probabilistic classifier based on Bayes Theorem, assumes feature independence

9. **Q: How does K-Means work?**
   **A:** Partitions data into K clusters based on nearest centroid

10. **Q: What is association rule mining?**
    **A:** Discovers patterns like "if X then Y" using support, confidence, lift
